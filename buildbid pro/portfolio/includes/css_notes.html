What is CSS

CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the presentation, styling, and visual formatting of an HTML document. While HTML provides the structure (the skeleton), CSS provides the looks (the clothes and decoration).

How Does CSS Work?

CSS works by targeting specific HTML elements and applying visual rules to them based on instructions defined in a stylesheet.

CSS Example


p { 
    font-family: Arial, sans-serif; 
    color: #444; 
} 

/* Class Selector: Targets any element with a specific class attribute (starts with a dot ".") */ 
.highlight { 
    background-color: yellow; 
    font-weight: bold; 
    padding: 5px; 
} 

/* ID Selector: Targets ONE unique element with a specific ID attribute (starts with a hash "#") */ 
#main-heading { 
    color: darkred; 
    text-align: center; 
    border-bottom: 3px solid darkred; 
    padding-bottom: 10px; 
}

Why is CSS important?

CSS is important because it allows developers to control the visual presentation of web pages separately from their structural content. It is what transforms a basic, plain text document into a visually appealing, branded, and functional website.

Common CSS Properties

Common CSS Properties

  • <color>: Sets the text color of an element (e.g., color: blue;).
  • <background-color>: Defines the background color of an element (e.g., background-color: #f0f0f0;).
  • <font-family>: Specifies the typeface used for text (e.g., font-family: Arial, sans-serif;).
  • <font-size>: Sets the size of the text (e.g., font-size: 16px;).
  • <text-align>: Controls the horizontal alignment of text within an element (e.g., text-align: center;).
  • <margin>: Defines the outer spacing (clears an area outside the border) of an element (e.g., margin: 10px;).
  • <padding>: Defines the inner spacing (clears an area inside the border, around the content) of an element (e.g., padding: 20px;).
  • <width> and <height>: Sets the dimensions of an element (e.g., width: 50%; or height: 100px;).
  • <border>: Sets the border around an element (e.g., border: 1px solid black;).
  • <display>: Specifies the type of box an element generates (e.g., display: block;, display: inline;, display: flex;).
  • <position>: Defines how an element is positioned on the page (e.g., position: static;, position: absolute;, position: fixed;).
  • <list-style>: Sets the style of list item markers (e.g., bullets or numbers) (e.g., list-style: none;).
