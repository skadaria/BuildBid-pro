What is HTML?

HTML, which stands for HyperText Markup Language, is the standard markup language used to create and structure content on the World Wide Web.

How Does HTML Work?

It works by using special labels, called "tags," to wrap around different parts of the content. These labels tell your web browser (like Chrome, Safari, or Firefox) what everything is and how to lay it out.

HTML Example


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Simple Page</title>
</head>
<body>
    <h1>Welcome</h1>
</body>
</html>

Why is HTML important?

HTML is important because it is the backbone of the entire World Wide Web. Without HTML, web browsers wouldn't know how to organize or display information, and we wouldn't have functional websites as we know them today.

Common HTML Elements

Common HTML Elements

  • <html>: The root element that wraps all content on the page.
  • <head>: Contains metadata about the document (not visible on the page).
  • <title>: Sets the title of the web page, displayed in the browser tab.
  • <body>: Contains all the visible content of the webpage.
  • <div>: A generic container used for grouping content for styling.
  • <h1> to <h6>: Define headings of different levels (most to least important).
  • <p>: Defines a paragraph of text.
  • <a>: The anchor tag used to create hyperlinks (href attribute is required).
  • <img>: Embeds an image in the document (src and alt attributes are required).
  • <ul> and <ol>: Define unordered (bulleted) and ordered (numbered) lists.
  • <li>: Defines a single list item within a list.
  • <span>: An inline container for grouping small pieces of text.
  • <button>: Defines a clickable button for user interaction.
