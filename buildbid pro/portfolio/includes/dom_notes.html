What is the DOM?

It is a programming interface (API) for HTML and XML documents. It represents the structure of a webpage as a tree of objects, allowing programs primarily JavaScript to access and change the content, structure, and style of a document dynamically.

How Does the DOM Work?

The DOM works by representing an HTML document as a hierarchical tree of objects that programming languages, primarily JavaScript, can interact with to modify the webpage dynamically.

DOM Tree Example


Document

    ├── Element: head

    └── Element: body
        ├── Element: h1
        │   └── Text: "Welcome"

            └── Text: "Hello world!"

Why is the DOM important?

The DOM is crucial because it acts as the bridge between a static web document (HTML) and dynamic programming languages (JavaScript), making modern, interactive websites possible.

Common DOM Methods

  • <document.getElementById()>: Selects a single element based on its unique ID attribute.
  • <document.querySelector()>: Selects the first element that matches a specified CSS selector (e.g., a class, ID, or element type).
  • <document.querySelectorAll()>: Selects all elements that match a specified CSS selector, returning a list of nodes.
  • <element.addEventListener()>: Attaches an event handler (like a 'click' or 'hover' function) to a specific element.
  • <element.appendChild()>: Adds a new node (element) as the last child of a parent element.
  • <document.createElement()>: Creates a new HTML element node that can then be added to the DOM.
  • <element.remove()>: Removes the selected element from the DOM structure.
  • <element.textContent>: Gets or sets the text content of an element and its descendants.
  • <element.innerHTML>: Gets or sets the HTML markup contained within an element (use with caution due to security risks).
  • <element.setAttribute()>: Sets the value of an attribute on the specified element (e.g., changing an image's `src`).
